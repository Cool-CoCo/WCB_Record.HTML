## ✨ 这是什么？

这是我在参与 [Web3 Career Build](https://web3career.build/) 暑期实习计划期间，
为自己搭建的一个本地网页形式的学习记录工具。

最开始只是想做一个能把「每天学了什么、做了什么」记清楚的页面，并且最好可以随时打开查看，
后来慢慢把三周的课程结构、任务清单、团队模板都整合了进去，
顺手加了一些方便自己用的交互功能，最终做成了现在这个样子。

页面只有一个 HTML 文件，打开浏览器就能用，不需要安装任何东西。

---

## 📐 页面功能

- **关卡折叠导航**：Week 1 / 2 / 3 和 Final Demo 区各是一个可展开 / 收缩的关卡卡片，Day 1–5 也支持单独折叠，方便聚焦当天任务。
- **暗黑 / 光明模式**：右上角一键切换，偏好保存在本地，刷新不还原。
- **进度追踪 HUD**：三个进度条实时显示每周完成天数，打完当天所有勾才算「过关」。
- **本地持久化**：所有填写内容和勾选状态自动保存到浏览器本地存储，不依赖服务器，关了再开也在。
- **角色分路径**：Week 2 每天的学习内容和资源，按 Research / Ops / Dev 三条路径切换展示。
- **像素像素还是像素**：低饱和度三色配色、像素点阵背景、2px 偏移阴影边框，主要是自己用着觉得也挺有意思。

## 📖 内容覆盖

| Week 1 | Web3 入门 & 开营 | 概念速记、钱包安全、测试网交易、合约部署、Monad 理解 |

## 🛠 使用方式

1. 下载 `web3-career-build-log.html`
2. 用浏览器打开（Chrome / Firefox / Edge 等 均可）
3. 按天填写，打勾记录，截图和链接粘贴进对应区域
4. 所有内容自动保存在本地，不需要登录或联网
