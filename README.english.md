# 🎮 Web3 Career Build — Level-by-Level Learning Log

> A pixel-art styled interactive learning log for the Web3 Career Build summer program,
> built to track three weeks of onchain learning, career path exploration, and team collaboration
> — the way you'd clear levels in a game.

---

## ✨ What is this?

This is a self-built local web tool I put together while taking part in the
[Web3 Career Build](https://web3career.build/) summer internship program.

It started as a simple way to keep track of what I was learning each day.
Over time, I added the full three-week course structure, daily task checklists,
team project templates, and a few interaction features I found genuinely useful.
The whole thing ended up as a single pixel-art HTML file.

It's a complete, self-contained pixel-style HTML page — with all content,
dark / light mode toggle, and collapsible level panels built in.
Just open `web3-career-build-log.html` in any browser. No install, no internet required.

---

## 🎮 Features

**Visual Style**
- Pixel grid background patterns (chain links / hex nodes / star dots — low-saturation, Telegram wallpaper inspired)
- Three hand-drawn pixel characters (Research / Ops / Dev roles) in the top banner
- Three-color palette only: `dark olive green / deep navy gray / warm cream` — intentionally low saturation
- All cards use 2px solid pixel borders with 4px offset shadows (retro game card aesthetic)

**⬛ / ⬜ Dark / Light Mode**
- Toggle button in the top-right corner (☀ / ☾)
- One click to switch; your preference is saved locally and persists on reload

**🎮 Collapsible Level Panels**
- Week 1 / 2 / 3 / Final Demo are each a collapsible level card — click the header to expand or collapse
- Each Day (Day 1–5) inside a week can also be independently toggled
- A level map at the top lets you jump directly to any section

**💾 Local Persistence**
- All text inputs and checkbox states are auto-saved to your browser's local storage
- Nothing is uploaded anywhere; your notes stay on your own device

**📊 Progress Tracking HUD**
- Three progress bars showing completion across Week 1 / 2 / 3
- A day counts as complete only when all its checkboxes are ticked

**🔤 Typography**
- `Courier New` monospace throughout for a pixel-native feel
- Base sizes at 10px / 12px; heading levels scale up proportionally

## 🛠 How to Use

1. Download `web3-career-build-log.html`
2. Open it in any browser (Chrome / Firefox / Edge all work fine)
3. Fill in your notes, check off tasks, paste links and screenshots into the relevant fields
4. Everything saves automatically in the browser — no login, no sync needed
